WEB - HTML

Hypertext Markup Language (HTML) is the standard mark up language for creating webpages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and orginally included cues for the appearance of the document.
